# CatBagShop (frontend Angular)

Ce dossier **`frontend/`** contient la landing **sac de douche pour chats** (inspirée de [demarky.store](https://sac-de-douche-pour-chats.demarky.store)), en **Angular 19**. Les prix affichés sont des **valeurs de démo** (TND), configurables dans `src/environments/environment.ts`.

Le **backend** (API Django) est dans `../backend/` — à brancher ensuite (formulaire commande, catalogue, etc.).

Plan d’ensemble du dépôt : [README-PLAN.md](../README-PLAN.md).

---

This project was generated using [Angular CLI](https://github.com/angular/angular-cli) version 19.0.5.

## Development server

À la racine du dépôt : `cd frontend`, puis :

```bash
npm install
npm start
```

(équivalent : `ng serve`.) Ouvrir `http://localhost:4200/`.

## Building

```bash
ng build
```

(exécuter depuis le dossier `frontend/`.)

## Running unit tests

```bash
ng test
```
